Этот проект содержит три функции:

1. `palindrome` - проверяет, является ли строка палиндромом.
2. `correct_number` - проверяет корректность номера мобильного телефона в России.
3. `punctuation` - удаляет знаки препинания из строки на русском языке.

## Установка

Установите необходимые зависимости через терминал:

pip install -r requirements.txt
